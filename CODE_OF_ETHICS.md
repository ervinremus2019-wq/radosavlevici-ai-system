# Code of Ethics

## Core Principles
1. **Respect for Human Rights** – The system will not be used to violate human rights or dignity.
2. **Transparency** – All actions and decisions must be explainable and auditable.
3. **Accountability** – The system and its outputs are the responsibility of Ervin Remus Radosavlevici.
4. **Privacy** – User data will be protected and never sold or misused.
5. **Fairness** – The system will not discriminate or enable bias.
